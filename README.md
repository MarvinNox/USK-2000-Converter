# MCS Zone Converter

Проєкт для роботи з українськими місцевими системами координат (МСК) та системами УСК-2000, з підтримкою перетворення координат у WGS84 для відображення на карті.
Проєкт в розробці, можливі неточності перетворень.

## Можливості

- Підтримка обласних МСК України.
- Перетворення координат через `proj4`.
- Відображення точок на карті за допомогою `Leaflet`.
- Використання централізованого списку зон `ZONES`.
- Підтримка точок у форматі `X/Y` з подальшим перерахунком у `lat/lon`.

## Структура даних

Основний конфігураційний масив:

```js
const ZONES = [
  {
    msk: "МСК-01",
    region: "Автономна Республіка Крим",
    epsg: 9831,
    lon0: 34.5,
    x0: 300000,
    y0: 0
  }
  // ...
];
```

Поля:

- `msk` — назва місцевої системи координат.
- `region` — область або місто застосування.
- `epsg` — код CRS.
- `lon0` — центральний меридіан.
- `x0`, `y0` — параметри зсуву системи.


## Примітка по координатах

У багатьох МСК/УСК-2000 набір координат може подаватися як `X/Y`, `northing/easting` або у зворотному порядку залежно від джерела. Перед перетворенням важливо перевірити, який саме порядок використано у вихідних даних.

## Джерела даних

- EPSG Registry.
- Переліки регіональних (місцевих) систем координат України.
- Документація про УСК-2000 та локальні системи координат.
